# Quest Sticky ToDo

ゲームのような直感操作を目指した、付箋ツリー型ToDoのプロトタイプです。

## できること

- 付箋をドラッグして移動
- 付箋下の丸い「＋」を引っ張って子タスクを作成
- 日付レーンへのスナップ
- 自動整列
- 作成日・目標日・完了日の記録
- 表示に使う日付の切り替え
- 履歴ログ
- localStorage保存

## 使い方

`index.html` をブラウザで開くだけです。

## 操作

- 付箋をドラッグ: 移動
- 付箋下の丸をドラッグ: 子タスクを生成
- 付箋をダブルクリック: タイトル編集
- 付箋を右クリック: 完了/未完了を切り替え
- 自動整列: ツリーを並べ直す
- 日付レーンOFF: 日付なしツリー表示
